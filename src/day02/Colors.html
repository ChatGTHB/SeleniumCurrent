Color
Each tile divided like a grid on the screen is called a PIXEL.
Resolution: how many boxes are there side by side,
             How many boxes are there under?
             that is, the number of pixels side by side and under each other.

1024 x 768
1920 x 1080  When the second one exceeds 1000, it is HD High Definition.


Each tile contains RED GREEN BLUE

It is called RGB COLOR system.
                  Red : R:99 G:0 B:0 -> Red light
                  Green : R:0 G:99 B:0 -> Green light
                  Blue : R:0 G:0 B:99 -> Blue light

RRGGBB  -> COLOR CODES consisting of 6 digits appear
000000 - FFFFFF  between codes

Red     -> FF0000  -> 0123456789ABCDEF
Green   -> 00FF00
Blue    -> 0000FF
Yellow  -> FFFF00
